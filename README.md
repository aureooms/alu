[js-integer](http://aureooms.github.io/js-integer)
==

Integer numbers library for JavaScript.

```js
integer.parse( 16 , 100 , 'ff' ) ; // [ 2 , 55 ]
integer.stringify( 100 , 16 , [ 2 , 55 ] ) ; // 'ff'
```
